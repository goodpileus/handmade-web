Accordion (responsiveness) (due 3/31)

Description

Create a website that uses media queries to construct a narrative across different screen sizes (or a browser that is being operated like an accordion...).

How might a website change when being viewed on a phone and not the desktop? How might the story it is telling change or adapt? Is it the same space or is it something entirely different?
