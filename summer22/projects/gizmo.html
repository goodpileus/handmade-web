Gizmo (javascript/interaction) (due 3/31)

Description

Create a website that changes/adapts through user interaction. Try to use javascript for this exercise but if you'd rather experiment with other methods of interaction that's fair game too!

Consider what interaction adds to the website; how it changes the visitor's experience and relationship with the site and its content. Try to give the interaction purpose and meaning relevant to the rest of the site.

Consider what differentiates a website from an app or software... Is this a meaningful distinction to draw?
