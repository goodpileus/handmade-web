a Small Web
(GDES 491, ST:Handmade Web)

June 8 - July 8
M-F, 1 - 2:50p

A summer class that will introduce the essential tools, techniques, and language of web design through a fast-paced but generous workshop model——the idea is to learn to code by treating it like any other material (sometimes you have to make a mess). We’ll learn how to surf, borrow, and write code ourselves. The course’s primary goal is to remove the barrier to entry for creative coding while examining the space of the web at large and questioning what a website is and can be.

The websites we will make may be small, but their hearts will be big.
