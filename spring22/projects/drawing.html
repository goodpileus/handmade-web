Drawing (flexbox/positioning) (due 3/3)

Description

Make a small website that experiments with using flexbox or other methods of positioning in some way. For this exercise, think less about the function of the website and more about the form... Use html and css to draw and construct a visual composition.

Maybe it's a still life, or a landscape, or a self-portrait, or a collage, or a pattern, or an expression of a feeling, or an interface of some kind (maybe familiar, maybe future or alien?), or something else. Just experiment and draw something with code!

Feel free to use a visual reference or craft the site freehand.
